指南：使用一键导入或手动方法将 VS Code 设置、扩展和配置文件迁移到 Cursor

Cursor 基于 VS Code 代码库，使我们能够专注于打造最佳的人工智能编码体验，同时保持熟悉的编辑环境。这使得将现有 VS Code 设置迁移到 Cursor 变得非常容易。

## 配置文件迁移

### 一键导入

以下是如何一键导入整个 VS Code 设置的方法：

1. 打开 Cursor 设置（⌘/Ctrl + Shift + J）
2. 导航至常规 > 帐户
3. 在 "VS 代码导入 "下，单击 "导入 "按钮

![vscode-import.webp](https://cursor.codekv.com/1746673770353.webp)

这将转移您的

- 扩展
- 主题
- 设置
- 按键绑定

### 手动配置文件迁移

如果您要在不同机器间移动，或希望对设置进行更多控制，可以手动迁移配置文件。

### 导出配置文件

1. 在 VS Code 实例中，打开命令调板 (⌘/Ctrl + Shift + P)
2. 搜索 "首选项：打开配置文件（UI）"
3. 在左侧边栏找到要导出的配置文件
4. 点击三点菜单并选择 "导出配置文件"
5. 选择导出到本地机器或 GitHub Gist

### 导入配置文件

1. 在 Cursor 实例中，打开命令调板 (⌘/Ctrl + Shift + P)
2. 搜索 "首选项：打开配置文件 (UI)"
3. 单击 "新建配置文件 "旁边的下拉菜单，然后单击 "导入配置文件"
4. 粘贴 GitHub Gist 的 URL 或选择 "选择文件"上传本地文件
5. 点击对话框底部的 "导入" 保存配置文件
6. 最后，在侧边栏选择新的配置文件，点击"√"图标将其激活

### 版本更新

## 版本更新

我们会定期将 Cursor 重定向到最新的 VS Code 版本，以保持功能和修复的最新性。为确保稳定性，Cursor 经常使用稍旧的 VS 代码版本。

### 活动栏方向

为了优化人工智能聊天界面的空间，我们将其设置为水平方向。如果你更喜欢垂直方向

1. 打开命令调板 (⌘/Ctrl + Shift + P)
2. 搜索 "首选项：打开设置（UI）"
3. 搜索`workbench.activityBar.orientation`
4. 将值设为`垂直`
5. 重启 Cursor
