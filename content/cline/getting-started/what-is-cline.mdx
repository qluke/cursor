---
title: 什么是 Cline？
---

Cline 是一款集成于 Microsoft Visual Studio Code 的 AI 开发助手。它为你的 IDE 和大语言模型（LLM）之间提供桥梁，助力代码开发，提高生产力，并降低新手入门门槛。根据权限设置，Cline 可以读取/写入文件、执行命令、使用你的网页浏览器，并通过 Model Context Protocol 服务器扩展能力。

Cline 的独特之处在于其对代码生成的深思熟虑和强大的集成能力。它不仅仅是生成代码片段，而是通过逐步规划方案、持续感知整个开发环境，并对所有更改都需明确批准，与开发者协作。Cline 能理解大型代码库，加速新工程师的上手流程，并通过 Model Context Protocol 市场连接数百种工具，实现从项目自动部署到自动化事件响应等多种能力——所有操作均可通过自然语言指令完成。
