---
title: 安装 Cline 
---


Cline 是一款 VS Code 扩展，将 AI 编程助手直接带到你的编辑器。你可以通过以下方式安装：

### 安装方式

- **VS Code 应用市场（推荐）：** 适用于标准 VS Code 和 Cursor 用户，最快捷。
- **Open VSX Registry：** 适用于 VSCodium 等兼容 VS Code 的编辑器。

### 🛠️ VS Code 应用市场：详细安装步骤

按照以下步骤快速安装并启用 Cline：

1. **打开 VS Code：** 启动 VS Code 应用。
    
    <Callout type="warn" title="注意">
    如果 VS Code 显示“正在运行扩展可能……”，请点击“允许”。
    </Callout>
2. **打开你的 Cline 文件夹：** 在 VS Code 中打开你在 Documents 下创建的 Cline 文件夹。
3. **进入扩展市场：** 点击侧边栏的扩展图标（`Ctrl + Shift + X` 或 `Cmd + Shift + X`）。
4. **搜索 'Cline'：** 在扩展搜索栏输入 `Cline`。

 
*VS Code 应用市场中 Cline 扩展准备安装*

1. **安装扩展：** 点击 Cline 扩展旁的"安装"按钮。
2. **打开 Cline：**
    - 点击侧边栏的 Cline 图标。
    - 或使用命令面板（`Ctrl/Cmd + Shift + P`），输入 "Cline: Open In New Tab" 获得更好的体验。
3. **故障排查：** 如果没有看到 Cline 图标，请尝试重启 VS Code。

<Callout type="info" title="专业建议">
你应该会在 VS Code 编辑器中看到 Cline 聊天窗口！
</Callout>

### 🌐 Open VSX Registry

适用于无法访问应用市场的 VS Code 兼容编辑器（如 VSCodium、Windsurf）：

1. 打开你的编辑器。
2. 进入扩展视图。
3. 搜索 "Cline"。
4. 选择 saoudrizwan 发布的 "Cline"，点击 **安装**。
5. 如有提示请重载编辑器。

### 👤 创建你的 Cline 账号

安装好 Cline 后，开始设置你的账号：

1. **登录 Cline：**
    - 点击 Cline 扩展中的 **登录** 按钮。
    - 页面会跳转到 [app.cline.bot](https://app.cline.bot/) 创建账号。
2. **免费体验额度：**
    - 无需信用卡即可开始！
3. **可用 AI 模型：**
    - Anthropic Claude 3.5-Sonnet（推荐用于编程）
    - DeepSeek Chat（高性价比选择）
    - Google Gemini 2.0 Flash
    - 以及更多——均可通过你的 Cline 账号使用。

### 💻 与 Cline 的第一次互动

你已经可以开始编程啦！将以下内容复制粘贴到 Cline 聊天窗口：

Copy`Hey Cline! Could you help me create a new project folder called "hello-world" in my Cline directory and make a simple webpage that says "Hello World" in big blue text?`

<Callout type="info" title="专业建议">
Cline 会帮你创建项目文件夹并搭建第一个网页！
</Callout>

### 🧩 使用 Cline 的小技巧

- **多提问：** 有疑问就问 Cline！
- **用截图：** Cline 能识别图片，展示你的工作进展。
- **复制粘贴报错：** 将错误信息粘贴到聊天框，Cline 会帮你解决。
- **用自然语言：** 用自己的话描述需求，Cline 会帮你转化为代码。
