# 🚀 开始之前

在正式编写代码前，请确保你已准备好以下基础工具：

1. **VS Code**

一款流行、免费且强大的代码编辑器。

📺 **推荐 YouTube 教程：** [如何安装 VS Code](https://www.youtube.com/watch?v=MlIzFUI1QGA)

<Callout type="info" title="专业建议">
在 macOS 上将 VS Code 安装到 Applications 文件夹，或在 Windows 上安装到 Program Files，方便从 Dock 或开始菜单快速启动。
</Callout>

###

2. **必备开发工具**

高效编程所需的基础软件：

- Homebrew (macOS)

- Node.js

- Git

👉 按照我们的详细指南，跟随 Cline 一步步安装必备开发工具。

📺 **推荐 YouTube 教程：**

*   **macOS:**
    
    *   [Install Homebrew on Mac](https://www.youtube.com/watch?v=hwGNgVbqasc)
        
    *   [Install Git on MacOS 2024](https://www.youtube.com/watch?v=B4qsvQ5IqWk)
        
    *   [Install Node.js on Mac (M1 | M2 | M3)](https://www.youtube.com/watch?v=I8H4wolRFBk)
        
    
*   **Windows:**

<Callout type="warn" title="注意">
如果安装过程中遇到权限问题，请尝试以管理员身份运行终端或命令提示符。
</Callout>

###

3. **整理你的项目**

在 Documents 文件夹下新建一个名为 `Cline` 的专用文件夹，用于存放所有编程项目：

- **macOS：** `/Users/[你的用户名]/Documents/Cline`
- **Windows：** `C:\Users\[你的用户名]\Documents\Cline`

在 `Cline` 文件夹内，建议清晰地组织各个项目：

- `Documents/Cline/workout-app` *(如：健身追踪应用)*
- `Documents/Cline/portfolio-website` *(如：个人作品集网站)*

<Callout title="小贴士">
从一开始就养成良好的项目整理习惯，能帮你节省大量时间，避免后续混乱！
</Callout>

###

4. **安装 Cline VS Code 扩展**

在 VS Code 内直接安装 Cline 扩展，提升你的编程效率：

- 参考 Cline 扩展安装教程

📺 **推荐 YouTube 教程：** [如何在 VS Code 安装扩展](https://www.youtube.com/watch?v=E7trgwZa-mk)

<Callout type="info" title="专业建议">
安装完成后请重启 VS Code，确保扩展被正确激活。
</Callout>

🎉 一切就绪！现在就开始用 **Cline** 更智能、更高效地编程吧。
