了解如何使用 Cursor 的闪光图标或快捷方式自动生成上下文相关的 Git 提交消息

Cursor 可以帮助您通过点击生成有意义的提交消息。以下是如何使用此功能：

1. 暂存你想要提交的文件
2. 在侧边栏中打开 Git 选项卡
3. 在提交消息输入框旁边寻找闪光（✨）图标
4. 点击闪光图标，根据你的暂存更改生成提交消息

生成的提交消息将基于您暂存文件中的更改和您仓库的 git 历史。这意味着 Cursor 将分析您当前的更改和之前的提交消息，以生成一个上下文适当的消息。Cursor 从您的提交历史中学习，这意味着如果您使用像 Conventional Commits 这样的约定，生成的消息将遵循相同的模式。
