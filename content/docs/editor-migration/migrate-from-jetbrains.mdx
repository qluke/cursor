Cursor 提供现代化的人工智能编码体验，可取代您的 JetBrains IDE。虽然过渡之初可能会感觉不同，但 Cursor 基于 VS Code 的基础提供了强大的功能和广泛的自定义选项。

## 编辑器组件

### 扩展

JetBrains 集成开发环境是很好的工具，因为它们已经为所使用的语言和框架进行了预配置。

Cursor 则不同，它是一个开箱即用的空白画布，你可以根据自己的喜好进行定制，而不受集成开发环境所适用的语言和框架的限制。

Cursor 拥有庞大的扩展生态系统，JetBrains IDE 提供的几乎所有功能（甚至更多！）都可以通过这些扩展重新创建。

下面就来看看其中一些流行的扩展：

### 键盘快捷键

Cursor 内置了键盘快捷键管理器，可将您喜爱的键盘快捷键映射到操作中。

有了这个扩展，你可以将几乎所有 JetBrains IDE 的快捷键直接引入 Cursor！请务必阅读该扩展的文档，了解如何根据自己的喜好进行配置：

## [IntelliJ IDEA 按键绑定](https://marketplace.visualstudio.com/items?itemName=k--kato.intellij-idea-keybindings)

[安装该扩展可将 JetBrains IDEs 的键盘快捷键引入 Cursor。](https://marketplace.visualstudio.com/items?itemName=k--kato.intellij-idea-keybindings)

### 主题

使用这些社区主题，在 Cursor 中重现您最喜爱的 JetBrains IDE 的外观和感觉。

您可以选择标准的 Darcula 主题，也可以选择与您的 JetBrains 工具的语法高亮相匹配的主题。

### 字体

为了让你的 JetBrains 体验更加完美，你可以使用官方的 JetBrains Mono 字体：

1. 下载并在系统中安装 JetBrains Mono 字体：
2. 安装字体后重启 Cursor
3. 打开 Cursor 中的设置 (⌘/Ctrl + ,)
4. 搜索 "字体家族
5. 将字体家族设置为`"JetBrains Mono`
