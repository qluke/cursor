# 安装

1. 访问[cursor.com](https://cursor.com/)，点击 "下载 "按钮 
    
    您操作系统的安装程序将自动下载
    
2. 运行安装程序并等待安装完成
3. 通过桌面快捷方式或应用程序菜单启动 Cursor

## 设置

首次启动时，系统会提示你配置一些设置，以确保你能快速上手并运行！

## 键盘快捷键

如果你来自不同的编辑器，可以选择默认快捷键，以便尽可能熟悉。

## 语言

如果您希望人工智能使用不同的语言与您交谈，可以输入您希望使用的语言名称。这可以在[规则](https://docs.cursor.com/context/rules)中进一步配置

## 代码库索引

Cursor 会在本地为你的代码库建立索引，以提供更好的人工智能建议。在[代码库索引](https://docs.cursor.com/context/codebase-indexing)中了解更多信息。

## CLI 快捷方式

你可以选择安装`游标`和`代码`命令，以便从终端启动 Cursor。

配置这些设置后，你将可以选择一键导入你的 VS Code 设置。如果你接受，这将把你的扩展、主题、用户设置和键盘快捷键导入到 Cursor 中，这样你就可以立即开始使用了。

接下来，系统会询问你的数据偏好。要了解更多信息并做出明智决定，请阅读我们的[隐私保护页面](https://docs.cursor.com/account/privacy)。

## 登录

1. 点击**"注册 "**或**"登录 "后，**系统会提示您设置账户。您可以选择使用自己的电子邮件，或使用 Google 或 GitHub 注册。
2. 登录后，你将被送回 Cursor，然后就可以**开始编码**了**！**

如果你是第一次使用 Cursor，注册后即可获得为期 14 天的 Cursor Pro 免费试用版。在我们的[网站](https://cursor.com/features)上了解有关 Cursor Pro 的更多信息。

## 从其他编辑器迁移

虽然 Cursor 与 VS Code 基于相同的核心构建，但仍有一些关键区别需要注意。此外，对于那些来自其他编辑器的用户来说，他们可能并不熟悉 Cursor 的结构。

为了帮助您开始使用，我们为您准备了一份指南，帮助您从其他编辑器迁移到 Cursor。
