## 前提条件

在开始之前，请确保您已

- 已安装[Python](https://python.org/)（建议使用 3.8 或更高版本）
- 用于版本控制的[Git](https://git-scm.com/)
- 已安装 Cursor 并更新至最新版本

## 框架支持

Cursor 可与流行的 Python 框架无缝协作：

- **网络框架**Django、Flask、FastAPI
- **数据科学**Jupyter、NumPy、Pandas
- **机器学习**TensorFlow、PyTorch、scikit-learn
- **测试**：pytest、unittest
- **应用程序接口**：请求、Aiohttp
- **数据库**SQLAlchemy, psycopg2
