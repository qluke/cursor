## Cursor 从何而来？

代码本质上是文本，我们编写代码的工具已经从简单的文本编辑器发展成为越来越智能的开发环

最初，我们有像语法高亮这样的功能，使代码更易读。然后，我们有像自动完成这样的功能，使代码更高效。

这些长期以来一直是标准，但有了 Cursor，我们正在重新发明你如何与代码互动的方式。

## 它是如何工作的？

Cursor 通过 LLMs（大型语言模型）的开发为用户提供了几个基本功能。
