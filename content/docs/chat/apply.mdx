了解如何使用 Cursor 的 Apply 功能从聊天中应用、接受或拒绝代码建议  

Cursor 的 Apply 允许你快速将聊天中的代码块建议集成到你的代码中。


## 应用如何工作

Apply 是一个我们专门开发的模型，旨在快速应用代码。它被设计为能够高效处理大规模代码变更，同时保持精确性和可靠性。

Apply 的主要优势在于其专注的目的 - 它不需要自己理解或生成代码。相反，它与您选择的 Cursor 模型协同工作，后者负责智能代码生成。Apply 然后接收这些更改，并在几秒钟内高效地将它们应用于数千行代码，使其非常适合大规模重构和更新。

## 应用代码块

要应用代码块建议，可以按下每个聊天代码块右上角的播放按钮。

这将编辑你的文件以纳入由 Chat 生成的代码。由于你在 Chat 中可以添加最多的上下文并与模型进行最多的互动，我们推荐使用 Chat + Apply 来进行更复杂的 AI 驱动代码更改。

## 接受或拒绝

一旦你应用了一个代码块，你可以浏览差异并接受或拒绝更改。你还可以在聊天代码块右上角点击“接受”或“拒绝”按钮。

`按 Ctrl/⌘ Enter 键`接受，`按 Ctrl/⌘ Backspace 键`拒绝。
