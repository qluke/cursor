通过 Cursor 的`应用`，您可以将聊天中的代码块建议快速集成到您的代码中。

## 应用如何工作

Apply 是我们专为快速代码应用而开发的专用模型。它旨在高效处理大规模代码更改，同时保持精确性和可靠性。

Apply 的关键优势在于其专注的目的--它本身不需要理解或生成代码。相反，它与您选择的 Cursor 模型协同工作，后者负责智能代码生成。然后，Apply 会接收这些更改，并在数秒内高效地将它们应用到数千行代码中，这使它成为大规模重构和更新的完美工具。

## 应用代码块

要应用代码块建议，可以按下每个聊天代码块右上角的播放按钮。

这将编辑你的文件，纳入聊天生成的代码。由于您可以在聊天中添加最多的上下文并与模型进行最多的来回切换，我们建议您使用聊天 + 应用来进行更复杂的人工智能驱动的代码更改。

## 接受或拒绝

应用代码块后，您可以查看差异并接受或拒绝更改。您也可以点击聊天代码块右上角的 "接受 "或 "拒绝 "按钮。

`按 Ctrl/⌘ Enter 键`接受，`按 Ctrl/⌘ Backspace 键`拒绝。
