## Ask 模式

Ask 模式允许您通过 AI 搜索和查询来探索和了解代码库，而无需进行任何更改。

Ask 是 Chat 的一种“只读”模式，用于提出问题、探索和学习代码库。它是 Cursor 的内置模式，并默认启用了搜索工具。

请阅读 [Chat 概览](#) 以了解 Cursor 中模式的工作原理。

---

### 与其他模式的区别

Ask 模式与其他默认模式（Agent 和 Manual）不同，因为它默认不会应用任何建议的更改——这使其成为一种“只读”模式，具有读取文件和其他上下文的全部能力，但无法自行进行任何更改。

这非常适合您不希望对代码库进行更改时使用，或者在实现解决方案之前与 AI 一起规划解决方案！

---

### 配置选项

- **模型**: 预先选择 Ask 模式的默认模型。  
- **快捷键**: 设置切换到 Ask 模式的快捷键。  
- **搜索代码库**: 允许 Cursor 自动搜索其上下文，而无需每次都手动使用 @ 文件来提供上下文供 AI 查看。
