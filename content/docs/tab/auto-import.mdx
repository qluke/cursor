---
title: 自动导入
description: Auto Import
---

 
在 TypeScript 和 Python（测试版）项目中，Tab 可以自动从项目的其他地方导入模块和函数，而无需手动键入导入语句。

只需开始使用现有文件中的方法，Tab 就会自动为你推荐导入语句。如果你接受，导入语句就会添加到你的文件中，而不会让你离开正在编写的代码。

## 疑难解答

如果您在使用自动导入时遇到问题，请确认您的项目语言是否有必要的扩展（如语言服务器），因为这是自动导入工作的必要条件。

您可以将Cursor移到尚未导入的函数或方法上，然后点击 ⌘ + .或 Ctrl + .，查看 "快速修复 "建议中是否建议导入 - 如果没有，则说明语言服务器没有工作。
