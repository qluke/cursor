了解 Tab 如何使用 AI 在编辑器中直接建议多行编辑、代码修复和上下文感知的补全

Cursor Tab 是我们原生的自动完成功能。它是一个更强大的 Copilot，能够建议整个 diffs，并且具有特别好的记忆力。

由自定义模型驱动的 Cursor Tab 可以：

- 在光标周围建议编辑，而不仅仅是插入额外的代码。
- 一次修改多行。
- 根据你最近的更改和代码检查错误提出建议。

免费用户可免费获得 2000 条建议。专业版和商业版可获得无限建议。

## 用户界面

当 Cursor 仅添加额外文本时，补全将显示为灰色文本。如果建议修改了现有代码，则会在当前行右侧以差异弹出窗口的形式显示。

您可以按`Tab 键`接受建议，或按`Esc` 键拒绝建议。要逐字部分接受建议，请按`Ctrl/⌘ →`。要拒绝建议，只需继续打字，或使用`Escape`取消/隐藏建议。

每按一次键或移动一次Cursor，Cursor 都会尝试根据你最近的更改提出建议。不过，Cursor 并不总是会显示建议；有时，模型已经预测到无需进行更改。

Cursor 可以在当前行的上方一行到下方两行之间进行更改。

## 切换

要打开或关闭该功能，请将鼠标悬停在应用程序右下方状态栏中的 "Cursor选项卡 "图标上。

## 键盘快捷键

从 "Cursor "菜单中选择 "设置">"键盘快捷方式"，然后搜索 "`接受Cursor切换建议`"，即可将 "Cursor切换 "绑定到自定义键盘快捷方式。

## 常见问题

### 在撰写评论时，Tab 碍事怎么办？

您可以在 "Cursor`设置`">"Tab Completion`完成`"中取消选中 "在注释中触发"，从而在注释中禁用 Cursor Tab。
