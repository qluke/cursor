编辑器

Cursor 中所有键盘快捷键的完整参考，包括聊天、Tab、终端和代码选择命令

Cursor 中键盘快捷键和按键绑定的高级概览。按`Cmd + R`，然后按`Cmd + S`，即可查看所有键盘快捷键。

要了解有关 Cursor 中键盘快捷键的更多信息，请查看《[VS 代码的按键绑定](https://code.visualstudio.com/docs/getstarted/keybindings)》，因为它是 Cursor 按键绑定的良好基准。

所有 Cursor 的按键绑定，包括 Cursor 特有功能的按键绑定，都可以在键盘快捷键设置页面中重新映射。

在 Windows 中，所有`Cmd`键都可以替换为`Ctrl`键。

## 一般设置

| 快捷键 | 操作 |
| --- | --- |
| Cmd + I (⌘I) | 切换副面板（除非与模式绑定） |
| Cmd + L (⌘L) | 切换副面板（除非与模式绑定） |
| Cmd + . (⌘.) | 打开模式菜单 |
| Cmd + / (⌘/) | 在 AI 模式之间循环 |
| Cmd + Shift + J (⌘⇧J) | 打开光标设置 |
| Cmd + , (⌘,) | 打开常规设置 |
| Cmd + Shift + P (⌘⇧P) | 打开命令调板 |

## 聊天

这些快捷键可在聊天输入框中使用。

| 快捷键 | 操作 |
| --- | --- |
| 输入 | 提交 |
| Cmd + Shift + Backspace (⌘⇧⌫) | 取消生成 |
| Cmd + L (⌘L)，选择代码 | 将选定代码添加为上下文 |
| Cmd + Shift + L (⌘⇧L)，选中代码 | 将选中的代码添加为上下文 |
| Cmd + Enter (⌘⏎) | 接受所有更改 |
| Cmd + Backspace (⌘⌫) | 拒绝所有更改 |
| 选项卡 | 循环到下一条信息 |
| Shift + Tab (⇧⇥) | 循环到上一条信息 |
| Cmd + Alt + / (⌘⌥/) | 打开模型切换 |
| Cmd + N (⌘N) / Cmd + R (⌘R) | 创建新聊天 |
| Cmd + Shift + K (⌘⇧K) | 以栏的形式打开作曲家 |
| Cmd + [ (⌘[) | 打开之前的聊天 |
| Cmd + ] (⌘]) | 打开下一个聊天 |
| Cmd + W (⌘W) | 关闭聊天 |
| Esc | 取消字段对焦 |

## Cmd+K

| 快捷键 | 操作 |
| --- | --- |
| Cmd+K | 打开 |
| Cmd + Shift + K | 切换输入焦点 |
| 输入 | 提交 |
| Cmd + Shift + Backspace (⌘⇧⌫) | 取消 |
| 选项 + Enter | 快速提问 |

## 代码选择和上下文

| 快捷键 | 操作 |
| --- | --- |
| @ | [@ 符号](https://docs.cursor.com/context/@-symbols) |
| # | 文件 |
| / | 快捷命令 |
| Cmd + Shift + L | 将选择添加到聊天 |
| Cmd + Shift + K | 在 "编辑 "中添加选区 |
| Cmd + L | 将选区添加到新聊天 |
| Cmd + M | 切换文件读取策略 |
| Cmd + → | 接受建议的下一个词 |
| Cmd + Enter | 在聊天中搜索代码库 |
| 选择代码，Cmd + C，Cmd + V | 添加复制的参考代码作为上下文 |
| 选择代码、Cmd + C、Cmd + Shift + V | 将复制的代码添加为文本上下文 |
