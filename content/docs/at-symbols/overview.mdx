## 概述

使用 @ 符号可以在 Cursor 中引用代码、文件、文档及其他上下文内容，用于聊天或操作。

在 Cursor 的输入框（如 **Composer**、**Chat** 和 **Cmd K**）中，您可以通过键入 **@** 符号触发操作。此时会弹出一个菜单，显示一系列建议项，并根据您的输入自动筛选最相关的内容。

---

### **键盘快捷键**

- 使用 **上下箭头键** 在建议列表中导航。  
- 按 **Enter** 键选择建议项。  
- 如果建议项是一个类别（如 **Files**），列表会过滤，仅显示该类别中最相关的内容。

---

### **可用 @ 符号列表**

以下是所有可用的 @ 符号及其功能：

- **@Files** - 引用项目中的特定文件  
- **@Folders** - 引用整个文件夹以提供更广泛的上下文  
- **@Code** - 引用代码库中的特定代码片段或符号  
- **@Docs** - 访问文档和指南  
- **@Git** - 访问 Git 历史记录和更改  
- **@Notepads** - 访问记事本  
- **@Past Chats** - 使用摘要化的会话记录  
- **@Cursor Rules** - 使用 Cursor 的规则  
- **@Web** - 引用外部网络资源和文档  
- **@Link (paste)** - 创建指向特定代码或文档的链接  
- **@Recent Changes** - 创建指向最近代码更改的链接  
- **@Lint Errors** - 引用 Lint 错误（仅限 Chat）  
- **@Definitions** - 查找符号定义（仅限 Cmd K）  

---

### **其他可用符号**

- **# Files** - 添加文件到上下文中，但不引用它们。  
- **/ Commands** - 将打开或活动的文件添加到上下文中。
