## @Web

使用 **@Web** 符号，支持动态搜索网络信息，将最近的在线内容和文档作为 Cursor 命令的上下文。

---

### **@Web 的功能**

通过 **@Web**，Cursor 会根据您提供的查询和上下文构建搜索请求，并在网络上查找相关信息，将其作为额外的上下文添加。

- **实时更新**：帮助 Cursor 获取最新的在线信息。  
- **快速检索**：允许 Cursor 在几秒内爬取多个网站，找到最相关的答案，无需用户手动搜索。

---

### **Agent 模式中的 @Web**

当使用 **Agent 模式**时，如果 Cursor 需要最新信息或额外的上下文，它会自动搜索网络。
