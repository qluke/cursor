## @Git

了解如何在 Cursor 的 Chat 中使用 **@Git** 符号引用 Git 提交记录、diffs 和 PRs，以分析更改并生成提交消息。

---

### **当前支持范围**

**@Git** 目前仅支持在 **Chat** 和 **Composer** 中使用。  
您可以通过 **@Git** 在 Chat 中将 Git 提交记录、diffs 或 Pull Requests 添加到提示中。

---

### **常见用例**

1. **扫描 diff 查找问题**  
   一个常见的使用场景是让 Cursor 的 AI 扫描 diff，寻找可能由这些更改引起的错误或问题。

2. **生成提交消息**  
   您可以使用 **@Diff of Working State** 根据当前的 diff 自动生成提交消息。
