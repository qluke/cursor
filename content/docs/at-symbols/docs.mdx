## @Docs

了解如何在 Cursor 中使用 **@Docs** 符号，将文档作为上下文，以及如何添加和管理自定义文档。

---

### **使用 @Docs**

Cursor 提供了一套已爬取、索引并准备好使用的第三方文档，您可以通过 **@Docs** 符号访问它们。  
您可以在这里查看默认预爬取文档的完整列表。

---

### **添加自定义文档**

如果您需要爬取和索引未包含在默认文档中的自定义文档，可以通过以下步骤完成：  
1. 输入 **@Docs > Add new doc**。  
2. 粘贴您需要索引的文档 URL，系统将弹出以下对话框：  

Cursor 会开始索引并学习该文档。之后，您可以像使用其他文档一样，将其用作上下文。  
**注意**：如果您希望索引所有子页面和子目录，请在 URL 后添加斜杠（/）。

Cursor 会自动保持文档索引的更新，并在文档被编辑或更改时周期性地重新索引。

---

### **管理自定义文档**

在 **Cursor Settings > Features > Docs** 中，您可以查看已添加的文档。  
在这里，您可以执行以下操作：  
- **编辑**文档信息  
- **删除**文档  
- **添加**新文档
