## 早期访问计划

了解如何加入 Cursor 的早期访问计划，测试实验性功能并提供反馈意见。

Cursor 提供了一个早期访问计划，允许您优先体验新功能和实验性功能。尽管这些功能令人兴奋，但它们可能比标准功能的稳定性稍差。

**注意**：Beta 功能是实验性的，可能包含错误或意外行为。如果您需要一个稳定的开发环境，建议保持标准设置。

---

### 加入早期访问计划

**要加入早期访问计划并接收预发布更新，请按照以下步骤操作：**

1. **打开 Cursor 设置**  
   在 macOS 上使用 **CMD+Shift+J**，在 Windows 和 Linux 上使用 **Ctrl+Shift+J** 打开 Cursor 的设置菜单。

2. **进入 Beta 菜单**  
   在设置侧边栏中找到并选择 **Beta 菜单**。

3. **选择“早期访问”选项**  
   找到 **“更新频率 (Update frequency)”** 下拉菜单，并选择 **“Early Access”** 以加入早期访问计划。

4. **等待下次更新**  
   等待下一次早期访问更新，它将自动推送到您的应用。

---

### 退出早期访问计划

如果您决定不再参与早期访问计划，可以通过以下步骤退出：  
1. 在设置菜单中将 **“更新频率 (Update frequency)”** 切换回 **“Standard”**。  
2. 等待下一次更新生效，或者从 [cursor.com](https://www.cursor.com) 重新下载最新版稳定版 Cursor。

---

### ⚠️ 注意事项

- Beta 功能是实验性的，可能包含错误或意外行为。  
- 使用早期访问版本的用户可能无法获得全面支持。  
- Beta 功能可能会在没有通知的情况下更改或移除，以便我们收集反馈并做出改进。
